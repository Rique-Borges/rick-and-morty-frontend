# Rick & Morty — Frontend

App React + TypeScript (Vite) para listar personagens.

Requisitos
- Node.js

Instalação & execução
- npm install
- npm run dev

Configuração
- Alterar API/backend em src/api/axios.ts

Estrutura rápida
- src/main.tsx, src/App.tsx
- src/pages, src/components, src/context, src/api
